<h2 align="center">小巧实用的前端工具类库。</h2>

## ✨ 特性

* ✏️ 使用TypeScript编写，类型友好
* ✨ 源于日常项目的积累，更实用
* 🐳 使用Jest进行严格测试，更稳定
* ⚡️ 支持按需加载，请配合babel-plugin-import使用

## 📦 安装

* npm安装

```
npm install @alitajs/autils --save
```

* yarn 安装(推荐)

```
yarn add @alitajs/autils
```
